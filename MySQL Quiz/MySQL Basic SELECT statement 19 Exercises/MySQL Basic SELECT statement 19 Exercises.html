## count(*)?

- 며칠 전에 DD님이 SH님께 설명하는 걸 얼핏 들었는데 정리를 해야 할 듯
- [count에 대한 설명](https://extbrain.tistory.com/54)

```
▶설명
테이블에 존재하는 데이터 갯수를 가져오고 싶을 때가 있습니다.
이 때 사용하는 함수가 COUNT 함수입니다.
COUNT 함수는 테이블에 컬럼의 데이터 갯수를 가져옵니다.
이 때 NULL인 데이터는 제외하고 계산합니다.
전체 행 갯수를 가져올 때는 컬럼 대신에 *를 지정합니다.

▶사용법
전체 행 갯수 가져오기
SELECT COUNT(*) FROM 테이블;

컬럼 데이터 갯수 가져오기
SELECT COUNT(컬럼) FROM 테이블;
```
    ## solution

select avg(SALARY) as avg_salary,
       count(distinct EMPLOYEE_ID) as count_employees
from employees


select avg(SALARY) as avg_salary,
       count(*) as count_employees
from employees

#	avg_salary	count_employees
1	6461.682243	107

  • 19. Write a query to get monthly salary (round 2 decimal places) of each and every employee, Note : Assume the salary field provides the 'annual salary' information.
  • Idea, Solution, Result
    ## Idea

- 일년 연봉이라고 가정했으니 나누기 12로 월급을 구한다.
